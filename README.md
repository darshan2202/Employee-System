# Employee Management System

 Created in Eclipse using Java.
 
 This application can be used to add and manipulate details of a employee. It can
 <ul>
 <li>create</li>
 <li>retrieve</li>
 <li>update</li>
 <li>delete data</li>
 </ul>
 
 Data is stored in MySql database in the form of table.
